--------------------------------
FILE ENCRYPT-DECRYPT by team-E10
--------------------------------

Executable file is on dist folder -> main.exe

Run it and choose an option.

If you want to encrypt a file, choose encrypt, 
enter the path of the file you want to encrypt,
a protection password and press done.
Along with the encrypted file there will be
a folder named test.txt. DON'T TOUCH IT, it's
important for the decryption to work.

If you want to decrypt a file, choose decrypt, 
enter the path of the file you want to decrypt,
the protection password you secured the file with, and press done.
The test.txt will be deleted and your file will have decrypted.
